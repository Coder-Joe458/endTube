# YouTube End of Video Recommendation Blocker

## Description

YouTube End of VideoRecommendation Blocker is a Chrome extension designed to enhance your YouTube viewing experience by removing distracting video and playlist recommendations. This extension allows users to focus on their intended content without being sidetracked by suggested videos or collections.

## Features

- Blocks YouTube's video and playlist recommendations on the video player
- Removes suggested videos and playlists from the sidebar while watching a video
- Hides end-of-video recommendations
- Improves focus and reduces time spent on unintended content
- Lightweight and easy to use

## Installation

1. Download the extension from the Chrome Web Store (link to be added once published)
2. Click "Add to Chrome" to install the extension
3. Once installed, you'll see the YouTube Recommendation Blocker icon in your Chrome toolbar

## Usage

1. The extension works automatically once installed
2. Visit YouTube.com and enjoy a distraction-free video watching experience
3. Both video and playlist recommendations will be hidden

## Technical Details

The extension works by:
- Targeting both video (.ytp-ce-element.ytp-ce-video) and playlist (.ytp-ce-element.ytp-ce-playlist) recommendation elements
- Hiding these elements and disabling their interactive features
- Using a MutationObserver to continuously check for and hide new recommendations as they appear

## Privacy

YouTube Recommendation Blocker respects your privacy. It does not collect or transmit any personal data. The extension only modifies the YouTube interface locally in your browser.
